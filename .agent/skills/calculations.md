# 計算邏輯與資料處理 (Skills)

本文件定義系統中負責資料轉換、計算與處理的程式邏輯說明。

## 1. 購物車金額計算 (Cart Calculation Skill)
- **輸入**：購物車內的商品陣列，包含 `{商品ID, 單價, 數量}`。
- **處理**：
  1. 遍歷購物車陣列。
  2. 針對每一項目執行 `小計 = 單價 * 數量`。
  3. 累加所有小計為 `總計金額`。
- **輸出**：回傳 `總計金額` 供前端顯示與結帳使用。

## 2. 訂單生成程序 (Order Generation Skill)
- **輸入**：會員 ID、購物車內容、總計金額。
- **處理**：
  1. 產生唯一訂單編號 (例如：以 timestamp 加上隨機碼 `ORD-1623456789-XXX`)。
  2. 記錄訂單建立時間。
  3. 將訂單狀態設為「已完成」(模擬環境)。
  4. 封裝成訂單物件並儲存至系統/資料庫(localStorage)。
- **輸出**：訂單物件，並清空目前的購物車。
