# 使用者互動與購衣流程 (Workflows)

本文件定義使用者在潮流服飾商城內的操作步驟與系統互動流程(SOP)。

## 1. 註冊與登入流程 (Auth Workflow)
1. 使用者點擊「會員」進入登入/註冊頁面。
2. 若無帳號，填寫帳號、密碼與手機進行註冊。
3. 系統依據 Rules 驗證資料，通過後寫入資料庫並自動登入。
4. 若已有帳號，輸入帳密登入，系統驗證成功後導向首頁。

## 2. 購物與結帳流程 (Purchase Workflow)
1. 使用者在首頁瀏覽商品列表。
2. 點選商品查看詳細資訊。
3. 輸入欲購買的「數量」，系統即時驗證數量不可為英文字母且不能超過庫存。
4. 點擊「加入購物車」，系統將商品與數量寫入購物車狀態。
5. (可重複上述步驟購買多種商品)
6. 進入「購物車」頁面，確認商品明細與總金額。
7. 點擊「結帳」，系統呼叫 Order Generation Skill 產生訂單並清空購物車。
8. 顯示「結帳成功」提示訊息，並導向歷史訂單頁面。

## 3. 歷史訂單查詢流程 (History Orders Workflow)
1. 使用者登入後，點選「歷史訂單」。
2. 系統讀取該使用者的歷史訂單紀錄。
3. 顯示訂單列表，包含訂單編號、日期、購買明細與總金額。
