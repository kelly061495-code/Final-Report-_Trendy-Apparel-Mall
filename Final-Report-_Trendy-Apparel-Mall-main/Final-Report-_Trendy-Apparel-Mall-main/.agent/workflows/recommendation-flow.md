# Workflow — 個人化智慧推薦

此流程描述了如何引導使用者完成個人化推薦並獲取建議商品。

## 交互流程
1. **觸發**: 使用者點擊首頁或 Shop 頁的「智慧推薦」按鈕。
2. **輸入資料**:
   - 身體數據: 身高、體重。
   - 預算限制: 願意支付的金額範圍。
   - 風格偏好: 勾選一個或多個標籤 (如 Street, Minimal)。
3. **運算分析**:
   - 系統計算 BMI。
   - 匹配對應的尺寸標準。
   - 根據風格與預算過濾 `PRODUCTS` 陣列。
4. **結果呈現**:
   - 顯示身形分析報告。
   - 推薦 1-4 個高度匹配的商品。
   - 提供直接開啟商品詳情的入口。
