# Skill — 訂單 CSV 匯出

此功能允許使用者將自己的歷史訂單記錄下載為本地 CSV 檔案, 方便記錄與對帳。

## 技術實現
1. **資料擷取**: 篩選出 `AppState.orders` 中屬於當前登入使用者的訂單。
2. **格式轉換**: 將訂單物件轉換為逗號分隔的字串格式 (CSV)。
3. **編碼處理**: 使用 `\ufeff` (BOM) 前綴, 確保 CSV 在 Excel 中開啟不會出現中文亂碼。
4. **下載觸發**: 創建一個虛擬的 `<a>` 標籤並觸發點擊下載。

## 欄位說明
- 訂單編號、日期、總金額、商品明細。

## 程式碼實作
- 邏輯內建於 `renderAccount()` 頁面的數據處理流程中 (或可擴展之 `exportOrdersCSV` 函式)。
