# Skill — 搜尋與篩選邏輯

商城提供高效的商品搜尋與分類篩選功能, 讓使用者能快速找到心儀的商品。

## 搜尋機制
- **關鍵字匹配**: 搜尋框會同時比對商品的 `name_zh` (中文名)、`name_en` (英文名) 以及 `style_tags` (風格標籤)。
- **不分大小寫**: 所有的匹配均會在轉為小寫後進行比較。
- **即時反饋**: 使用者按下 Enter 後, 會立即導向 Shop 頁面並展示符合條件的商品。

## 分類篩選
- 使用者可以透過導覽列或 Shop 頁面的切換按鈕進行分類過濾。
- 分類包含：全部、上衣、褲子、連帽衫、外套、包包、配飾。

## 程式碼實作
- 搜尋邏輯：`index.html` 中的 `renderShop()`。
- 事件監聽：`index.html` 中的 `DOMContentLoaded` 回呼函式。
