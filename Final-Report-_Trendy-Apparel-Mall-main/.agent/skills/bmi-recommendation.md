# Skill — 智慧推薦系統

系統提供個人化的穿搭與尺寸建議功能，透過簡單的身體數據分析為使用者尋找最適合的商品。

## 推薦邏輯 (本地演算法)
1. **BMI 計算**: 使用公式 `體重 / (身高/100)^2`。
2. **尺寸建議**:
   - BMI < 18.5: 建議 XS/S 或 28 碼。
   - BMI 18.5 - 24: 建議 M 或 30/32 碼。
   - BMI > 24: 建議 L/XL 或 34/36 碼。
3. **風格過濾**: 結合使用者選取的風格標籤 (如 Street, Techwear) 篩選產品庫。
4. **預算篩選**: 若使用者有輸入預算，則過濾掉超過金額的單品。

## UI 互動
- 使用者點擊首頁「智慧推薦」按鈕開啟面板。
- 輸入數據後，畫面會動態生成推薦理由及建議清單。

## 程式碼位置
- 邏輯實現：`index.html` 中的 `generateRecommendations()`。
