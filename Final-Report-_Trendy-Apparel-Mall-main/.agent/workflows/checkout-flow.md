# Workflow — 結帳流程

本流程描述使用者從選購商品到完成訂單的標準化步驟。

## 流程步驟
1.  **進入結帳**: 使用者在購物車抽屜點擊「前往結帳」。若未登入, 系統將引導至登入/註冊頁面。
2.  **填寫收件資訊**:
    *   姓名: 收件人真實姓名 (至少 2 字)。
    *   Email: 預設帶入會員帳號, 可修改。
    *   聯絡電話: 必須符合 09xx 格式。
    *   地址與地區: 選擇配送縣市並填寫門牌號碼。
    *   郵遞區號: 3 或 5 位數驗證。
3.  **套用優惠**:
    *   輸入代碼 (如 `WELCOME10`, `SAVE300`)。
    *   系統即時檢核門檻並重新計算總額。
4.  **訂單總覽**: 確認商品清單、折扣金額、運費 (滿 2000 免運) 以及最終付款金額。
5.  **送出訂單**: 點擊「確認訂單並送出」, 系統會將訂單寫入 `localStorage` 的 `orders` 陣列中。
6.  **完成跳轉**: 導向 `order-confirmed` 成功頁面, 展示勾選圖示與回首頁按鈕。

## 異常處理
- 若表單未填寫完全, 將彈出 Toast 提示错误。
- 若購物車為空, 將自動跳轉回首頁。
