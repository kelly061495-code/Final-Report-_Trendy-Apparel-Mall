# Knowledge — 尺寸系統

記錄商城各類產品的規格定義與試算標準。

## 1. 上衣 (Tops)
- **XS**: 胸寬 50, 衣長 64
- **S**: 胸寬 53, 衣長 67
- **M**: 胸寬 56, 衣長 70
- **L**: 胸寬 59, 衣長 73
- **XL**: 胸寬 62, 衣長 76
- **XXL**: 胸寬 65, 衣長 79

## 2. 褲裝 (Pants)
- **28**: 腰圍 71
- **30**: 腰圍 76
- **32**: 腰圍 81
- **34**: 腰圍 86
- **36**: 腰圍 91

## 3. 配飾 (Accessories)
- 統一採用 **One Size** (均碼)。

## 4. BMI 數據對照 (智慧推薦用)
- **過輕 (< 18.5)**: 建議 XS/S。
- **正常 (18.5 - 24)**: 建議 M。
- **過重/肥胖 (> 24)**: 建議 L/XL。
