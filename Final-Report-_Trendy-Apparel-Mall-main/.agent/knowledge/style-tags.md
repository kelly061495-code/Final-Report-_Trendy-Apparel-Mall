# Knowledge — 風格標籤

此文件定義了商品庫中使用的風格標籤, 這些標籤用於導航與智慧推薦過濾。

## 標籤列表
- **Street**: 街頭風格, 主打寬鬆與態度。
- **Oversized**: 寬鬆剪裁, 當前潮流主流。
- **Dark**: 暗黑系, 以黑色調與特殊印花為主。
- **Techwear**: 機能風, 強調防水與多口袋設計。
- **Cyber**: 賽博龐克, 具備未來感設計。
- **Minimal**: 極簡派, 純色、無大 Logo。
- **Y2K**: 千禧復古風, 色彩繽紛或具備 2000 年代設計元素。
- **Punk**: 龐克風, 包含皮衣、鏈條與叛逆元素。
- **Korean**: 韓系風格, 質感與都會剪裁的主流設計。

## 應用
- 在 `products-data` 中作為商品的 `style_tags` 屬性。
- 在「智慧推薦」面板中作為使用者的偏好勾選項。
