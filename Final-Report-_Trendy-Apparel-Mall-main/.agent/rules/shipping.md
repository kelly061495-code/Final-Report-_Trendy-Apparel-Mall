# Rule — 運費規則

系統根據消費金額自動判斷是否符合免運標準。

## 運費計算
- **固定運費**: 每筆訂單預設 NT$ 150。
- **免運門檻**: 單筆訂單小計 (不含折扣) 滿 **NT$ 2,000** 時，運費自動變為 $0。
- **免運優惠券**: 
  - 若使用者套用 `FREESHIP` 優惠碼。
  - 系統會自動抵扣運費 (最高抵扣 $150)。

## 呈現方式
- 結帳摘要中會明確列出「運費」一欄。
- 若符合免運，金額會顯示為綠色的「免運費」。

## 程式碼位置
- 計算位置：`index.html` 中的 `renderCheckout()` 內的變數 `ship`。
