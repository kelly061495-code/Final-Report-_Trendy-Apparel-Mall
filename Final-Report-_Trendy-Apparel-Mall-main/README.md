# TRENDY APPAREL MALL — 潮流服飾商城整合實務

## 專案簡介

TRENDY APPAREL MALL 是一套以極簡街頭風格為核心的 Standalone 電商應用系統。使用者可透過響應式網頁介面瀏覽 24 件精選潮流單品、加入購物車並完成結帳，系統內建完整的會員成長體系與智慧穿搭推薦功能。本專案採單一檔案架構開發，將原本複雜的 JavaScript 邏輯與 Tailwind CSS 整合，並利用 Browser LocalStorage 實現數據持久化，無需後端資料庫即可運作。系統深度結合 Antigravity AI 代理人架構，整合 `.agent` 知識庫與規則定義，實現從商品展示、BMI 風格診斷到訂單細節查詢的全流程自動化體驗，完美呈現現代精品電商的高階視覺與流暢互動。

**主要功能：**
- 👕 24 件精品型錄瀏覽與多視角詳細資料頁
- 🛒 智慧購物車管理與多品項多數量結帳
- 💎 進階會員階級系統 (Star/Silver/Gold/Diamond)
- 🧪 智慧 BMI 穿搭診斷與風格分析推薦
- 🔍 歷史訂單細節查詢 (含商品圖、尺寸、單價)
- 🗓️ 預購商品分類模組與到貨日提醒系統
- 📱 全響應式 Glassmorphism (玻璃擬態) 視覺設計
