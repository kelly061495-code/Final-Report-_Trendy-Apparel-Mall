## 二、 核心架構：單文件電商應用 (Single-file App)

配合期末交付需求,本專案已將原本分散的 18 個 JavaScript 模組與 CSS 樣式完整整合進單一檔案 `index.html`。這種設計具有以下優勢:
1. **即點即用**: 無需安裝 Node.js、Python 或任何後端伺服器,雙擊檔案即可在瀏覽器開啟。
2. **純淨無依賴**: 徹底移除所有外部品牌標誌、第三方追蹤腳本及複雜的後端伺服器依賴。
3. **高效本地運作**: 所有的商業規則 (優惠券、運費計算、BMI 推薦) 均已轉為本地 JavaScript 演算法實作。

## 三、 已實作的核心功能

### 1. 產品模組 ✅
- **全動態渲染**: 依分類 (上衣、褲子、外套、包包、配飾) 自動生成商品列表。
- **預購系統**: 特殊預購商品識別,畫面標註預購標籤,並依到貨日自動試算。
- **智慧搜尋**: 即時搜尋功能,支援商品名稱關鍵字匹配。

### 2. 採購與結帳模組 ✅
- **多功能購物車**: 支援跨頁面加入商品、即時調整數量、持久化存儲 (localStorage)。
- **完整結帳流程**:
  - **配送資訊**: 收件人、電話 (09 格式)、地址、郵遞區號、期望配送日驗證。
  - **優惠碼系統**: 支援 `WELCOME10`、`SAVE300`、`FREESHIP` 等多種折扣類型。
  - **金額自動試算**:
    - 滿 NT$ 2,000 自動免運。
    - 支援折扣碼與免運券併用計算。
    - 訂單摘要顯示現貨與預購拆分詳情。

### 3. 會員與訂單管理 ✅
- **本地帳戶系統**: 註冊、登入、登出功能。
- **歷史訂單查詢**: 記錄個人過往所有購買記錄,隨時查看訂單總額與日期。

## 五、 使用方法

1.  **直接開啟**: 在瀏覽器中直接開啟 `index.html`。
2.  **連網環境**: 由於商品圖片使用 Unsplash 高畫質圖庫,請確保電腦處於連網狀態以獲得完整視覺體驗。
3.  **無後端需求**: 本版本已取消所有後端 API 依賴,所有推薦演算法與數據處理均在瀏覽器內完成。
